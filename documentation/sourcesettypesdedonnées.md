# sources et types de données

Notre infrastructure à pour but de simuler le réseau de dunder mifflin. qui est composé de server Windows et Linux ainsi que des postes utilisateurs Windows
Nous cherchons à simuler les données que pourrait produire ces machines

## Windows

nos machines windows utilisent sysmon ainsi que les logs windows afin de remonter les informations sur les événemments de ces dernières.

## Debian

notre server debian 13 utilise kunai pour renvoyer les informations des événement du systeme.
